## Navigation
- `Ctrl` + `P` : Quickly open a file
- `Ctrl` + `Shift` + `O` : Go to a symbol in the file
- `Ctrl` + `G` : Go to a specific line
- `Ctrl` + `Tab` : Switch between open files

## Editing
- `Ctrl` + `C` / `Ctrl` + `X` / `Ctrl` + `V` : Copy / Cut / Paste
- `Ctrl` + `/` : Comment / Uncomment a line
- `Alt` + `Up` / `Alt` + `Down` : Move the line up / down
- `Ctrl` + `D` : Select next matching word
- `Ctrl` + `Shift` + `L` : Select all occurrences

## Search
- `Ctrl` + `F` : Search within the file
- `Ctrl` + `H` : Replace within the file
- `Ctrl` + `Shift` + `F` : Search in the entire project
- `Ctrl` + `Shift` + `H` : Replace in the entire project

## Terminal
- `Ctrl` + `` ` `` : Open / close the integrated terminal
- `Ctrl` + `Shift` + `` ` `` : Create a new terminal

## Interface
- `Ctrl` + `B` : Hide / show the sidebar
- `Ctrl` + `Shift` + `E` : Explorer / File view
- `Ctrl` + `Shift` + `G` : Version control (Git)
- `Ctrl` + `Shift` + `D` : Debug
- `Ctrl` + `Shift` + `X` : Extensions

## Command Palette
- `Ctrl` + `Shift` + `P` or `F1` : Open the command palette
